# ALIGNMENT - 求职追踪应用

## 📋 项目概述

**项目名称**: Job Tracker (求职追踪器)
**项目类型**: 前后端分离 Web 应用
**创建时间**: 2026-03-11

---

## 🎯 核心需求

### 1. 功能需求

#### 1.1 投递记录管理
- 记录投递的公司名称
- 记录申请的职位名称
- 记录公司招聘官网 URL
- 记录投递时间
- 支持增删改查操作

#### 1.2 流程追踪
- **状态管理**：投递中 → 简历筛选 → 面试中 → 已offer → 已拒绝
- **可视化看板**：按状态展示所有投递记录
- **时间线**：展示每个投递的完整历程
- **面试日历**：记录面试时间并提醒

#### 1.3 AI 辅助功能
- **职位描述优化**：根据公司名和职位自动生成/优化职位描述
- **求职信生成**：根据职位要求自动生成针对性求职信
- **面试准备**：根据职位生成可能的面试问题和参考答案
- **集成方式**：通过 LM Studio API (本地 Gemma3-4b 模型)

#### 1.4 面试答辩记录
- 记录面试日期
- 记录面试形式（视频/现场/电话）
- 记录面试官信息
- 记录面试问题与回答
- 记录面试复盘和总结

#### 1.5 数据管理
- 导出数据（CSV/JSON）
- 数据备份功能
- 统计分析（投递数量、转化率等）

### 2. 非功能需求

- **性能要求**：页面加载时间 < 2s
- **可用性**：支持离线使用（AI 功能除外）
- **安全性**：无需用户认证（单机使用）
- **兼容性**：支持主流现代浏览器

## 🔧 技术栈约定

### 前端
| 类别 | 技术 | 版本要求 |
|------|------|----------|
| 框架 | React | 18+ |
| 构建工具 | Vite | 最新版 |
| UI 框架 | Tailwind CSS + shadcn/ui | 最新版 |
| 状态管理 | Zustand | 最新版 |
| HTTP 客户端 | Axios | 最新版 |
| 路由 | React Router | v6+ |

### 后端
| 类别 | 技术 | 版本要求 |
|------|------|----------|
| 框架 | Spring Boot | 3.x |
| Java 版本 | JDK | 17+ |
| ORM | MyBatis Plus | 最新版 |
| 数据库 | MySQL | 8.0+ |
| API 规范 | RESTful | - |

### AI 集成
- **模型**: Gemma3-4b (通过 LM Studio)
- **API 规范**: OpenAI 兼容接口
- **本地地址**: http://localhost:1234/v1
