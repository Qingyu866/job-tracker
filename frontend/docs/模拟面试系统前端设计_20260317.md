**模拟面试与 AI 聊天助手是两个完全独立的功能模块**

**核心区别**：

| 特性 | AI 聊天助手 | 模拟面试 |
|------|-----------|---------|
| **用途** | 通用 AI 助手，帮助管理求职 | 针对特定职位的模拟面试 |
| **页面** | 内嵌在现有页面中 | 独立的全屏面试页面 |
| **对话方** | 单一 AI 助手 | 三 AI 面试官（主/副/评审） |
| **对话内容** | 聊天历史持久化 | 面试会话独立隔离 |
| **进入方式** | 点击聊天图标 | 从职位详情页进入 |

## 一、功能概述

### 1.1 用户场景

```
场景1：发起模拟面试
用户: 在职位详情页点击"模拟面试"按钮
系统: 跳转到独立的模拟面试页面（新窗口或新路由）
     基于职位 JD + 用户简历创建面试会话
AI 面试官: "你好！我是今天的面试官，我看到你应聘的是字节跳动的Java后端..."
            "看到你简历上写了电商秒杀项目，能详细讲讲高并发怎么处理的吗？"

场景2：面试进行中
用户: [在面试页面输入回答]
系统: 副面试官决定下一主题 → 主面试官生成新问题 → 用户继续回答
循环：直到达到设定轮数或用户主动结束

场景3：查看面试报告
用户: 结束面试后查看评估报告
系统: 显示总分、各轮评分、简历真实性分析、改进建议
```

### 1.2 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 独立面试页面 | 全屏专用面试界面，与聊天助手分离 | P0 |
| 发起面试 | 从职位详情页进入，选择简历后开始 | P0 |
| 面试对话 | 与三 AI 面试官实时对话 | P0 |
| 进度跟踪 | 显示当前轮次、已考察技能 | P1 |
| 结束面试 | 生成面试报告 | P1 |
| 历史记录 | 查看过去的面试会话 | P2 |

### 1.2 核心功能

### 3.2 InterviewChat 组件

```tsx
interface InterviewChatProps {
  messages: InterviewMessage[];
  onSendMessage: (content: string) => void;
  disabled?: boolean;
}

function InterviewChat({ messages, onSendMessage, disabled }: Props) {
  const [input, setInput] = useState('');
  const chatEndRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    chatEndRef.current?.scrollIntoView({ behavior: 'smooth' });
  }, [messages]);

  return (
    <div className="interview-chat">
      <div className="messages-container">
        {messages.map((message) => (
          <ChatMessage
            key={message.id}
            message={message}
            isAI={message.role === 'ASSISTANT'}
          />
        ))}
        <div ref={chatEndRef} />
      </div>

      <div className="chat-input-area">
        <TextArea
          value={input}
          onChange={(e) => setInput(e.target.value)}
          placeholder="输入你的回答..."
          disabled={disabled}
          onKeyDown={(e) => {
            if (e.key === 'Enter' && !e.shiftKey) {
              e.preventDefault();
              handleSend();
            }
          }}
        />
        <Button
          onClick={handleSend}
          disabled={!input.trim() || disabled}
        >
          发送
        </Button>
      </div>
    </div>
  );
}
```

### 3.3 SkillProgressTracker 组件

```tsx
interface SkillProgressTrackerProps {
  skillsCovered: SkillCovered[];
  skillsPending: SkillPending[];
}

function SkillProgressTracker({
  skillsCovered,
  skillsPending
}: SkillProgressTrackerProps) {
  const coveragePercentage = useMemo(() => {
    const total = skillsCovered.length + skillsPending.length;
    return total > 0 ? Math.round((skillsCovered.length / total) * 100) : 0;
  }, [skillsCovered, skillsPending]);

  return (
    <div className="skill-progress-tracker">
      <div className="progress-header">
        <span>技能覆盖进度</span>
        <Badge>{coveragePercentage}%</Badge>
      </div>

      <Progress value={coveragePercentage} className="progress-bar" />

      <div className="skills-list">
        <div className="covered-section">
          <h4>已考察 ({skillsCovered.length})</h4>
          <div className="skill-chips">
            {skillsCovered.map((skill) => (
              <Badge key={skill.id} variant="secondary">
                {skill.name}
              </Badge>
            ))}
          </div>
        </div>

        <div className="pending-section">
          <h4>待考察 ({skillsPending.length})</h4>
          <div className="skill-chips">
            {skillsPending.map((skill) => (
              <Badge key={skill.id} variant="outline">
                {skill.name}
              </Badge>
            ))}
          </div>
        </div>
      </div>
    </div>
  );
}
```

```typescript
// src/services/interviewApi.ts
export const interviewApi = {
  /**
   * 创建面试会话
   */
  async startInterview(applicationId: number, userId: number): Promise<MockInterviewSession> {
    return request.post('/mock-interview/start', {
      applicationId,
      userId
    });
  },

  /**
   * 获取会话详情
   */
  async getSession(sessionId: string): Promise<MockInterviewSession> {
    return request.get(`/mock-interview/sessions/${sessionId}`);
  },

  /**
   * 发送消息（用户回答）
   */
  async sendMessage(sessionId: string, content: string, roundNumber: number): Promise<InterviewMessage> {
    return request.post(`/mock-interview/sessions/${sessionId}/message`, {
      content,
      roundNumber
    });
  },

  /**
   * 获取会话消息列表
   */
  async getMessages(sessionId: string): Promise<InterviewMessage[]> {
    return request.get(`/mock-interview/sessions/${sessionId}/messages`);
  },

  /**
   * 结束面试
   */
  async finishInterview(sessionId: string): Promise<MockInterviewSession> {
    return request.post(`/mock-interview/sessions/${sessionId}/finish`);
  },

  /**
   * 获取会话评分列表
   */
  async getEvaluations(sessionId: string): Promise<MockInterviewEvaluation[]> {
    return request.get(`/mock-interview/sessions/${sessionId}/evaluations`);
  },

  /**
   * 获取用户的面试会话列表
   */
  async getUserSessions(userId: number): Promise<MockInterviewSession[]> {
    return request.get(`/mock-interview/sessions/user/${userId}`);
  }
};
```

```typescript
   * 获取用户的所有简历
   */
  async getUserResumes(userId: number): Promise<UserResume[]> {
    return request.get(`/resumes/user/${userId}`);
  },

  /**
   * 获取用户的默认简历
   */
  async getDefaultResume(userId: number): Promise<UserResume> {
    return request.get(`/resumes/user/${userId}/default`);
  },

  /**
   * 设置默认简历
   */
  async setDefaultResume(resumeId: number, userId: number): Promise<void> {
    return request.put(`/resumes/${resumeId}/default`, null, {
      params: { userId }
    });
  },

  /**
   * 获取简历的项目经历
   */
  async getProjects(resumeId: number): Promise<ResumeProject[]> {
    return request.get(`/resumes/${resumeId}/projects`);
  },

  /**
   * 获取简历的技能
   */
  async getSkills(resumeId: number): Promise<ResumeSkill[]> {
    return request.get(`/resumes/${resumeId}/skills`);
  }
```

## 七、用户体验流程

### 7.1 发起面试流程

```
用户操作流程：
1. 在职位详情页点击"模拟面试"按钮
2. 弹出简历选择对话框
3. 选择要使用的简历（可预览）
4. 确认面试信息（公司、职位、简历）
5. 点击"开始面试"
6. 进入面试对话页面

异常处理：
- 没有简历 → 提示先创建简历
- 有多份简历 → 显示列表让用户选择
- 创建会话失败 → 显示错误信息
```

### 7.2 面试进行流程

```
用户操作流程：
1. 主面试官提问
2. 用户阅读问题，组织回答
3. 输入回答并发送
4. 副面试官分析回答，决定下一主题
5. 主面试官根据主题生成新问题
6. 循环直到面试结束

状态反馈：
- 显示当前轮次：2 / 5
- 显示已考察技能
- 显示待考察技能
- 显示预计剩余时间
```
