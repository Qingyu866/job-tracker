# 前端 API 修改指南

**后端已完成重构，移除了 `/data/*` 路径前缀**

## 📝 修改示例

### 示例 1: 基本修改

```typescript
// ❌ 旧代码
apiClient.get('/data/applications')
apiClient.get('/data/interviews')
apiClient.get('/data/companies')

// ✅ 新代码
apiClient.get('/applications')
apiClient.get('/interviews')
apiClient.get('/companies')
```

### 示例 2: 特殊路径

```typescript
// ❌ 旧代码
apiClient.get('/data/applications/search')
apiClient.get('/data/applications/123/interview-progress')
apiClient.get('/data/applications/123/detail')

// ✅ 新代码
apiClient.get('/search/applications')
apiClient.get('/interviews/applications/123/progress')
apiClient.get('/export/applications/123/detail')
```

## ⚠️ 其他接口（无需修改）

以下接口**没有变化**，不需要修改：

- `/api/auth/*` - 认证相关
- `/api/chat/*` - 聊天相关
- `/api/status/*` - 状态转换
- `/api/resumes/*` - 简历管理
- `/api/skills/*` - 技能标签
- `/api/ocr/*` - OCR 识别
- `/api/mock-interview/*` - 模拟面试

## 🔧 快速替换

使用编辑器的"查找替换"功能：

**查找**: `/api/data/`
**替换为**: `/api/`

**特殊情况**（需要单独处理）：
- `/api/data/applications/search` → `/api/search/applications`
- `/api/data/applications/{id}/interview-progress` → `/api/interviews/applications/{id}/progress`
- `/api/data/applications/{id}/current-interview` → `/api/interviews/applications/{id}/current`
- `/api/data/applications/{id}/detail` → `/api/export/applications/{id}/detail`
- `/api/data/applications/{id}/logs` → `/api/logs/applications/{id}`
