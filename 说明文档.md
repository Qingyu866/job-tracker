## 📋 项目概述

这是一个帮助求职者管理和追踪求职进度的 Web 应用，支持投递记录管理、流程追踪、AI 辅助生成等功能。

### 核心特性

- ✅ **投递管理**：记录公司、职位、薪资等信息
- ✅ **进度追踪**：可视化看板展示求职进度
- ✅ **AI 辅助**：集成本地 Gemma3-4b 模型优化求职材料
- ✅ **面试记录**：详细记录面试过程和反馈
- ✅ **数据分析**：统计投递转化率等指标

---

## 🛠️ 技术栈

### 前端
- **框架**：React 18 + Vite
- **UI**：Tailwind CSS + shadcn/ui
- **状态**：Zustand
- **HTTP**：Axios

### 后端
- **框架**：Spring Boot 3.x
- **数据库**：MySQL 8.0+
- **ORM**：MyBatis Plus
- **JDK**：17+

### AI 集成
- **模型**：Gemma3-4b
- **平台**：LM Studio
- **接口**：OpenAI 兼容 API

## 🚀 快速开始

### 环境要求

- Node.js 18+
- JDK 17+
- MySQL 8.0+
- LM Studio（运行 Gemma3-4b）

### 后端启动

```bash
cd backend
mvn spring-boot:run
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 数据库初始化

```bash
mysql -u root -p < docs/init.sql
```
