## 一、功能概述

### 1.1 用户场景

```
场景1：聊天中发送图片
用户: [上传简历图片]
AI: "我看到这是一份简历，让我帮你解析..."
    [调用 OCR 工具识别]
AI: "识别完成！这份简历显示：3年Java后端经验，熟悉Spring Boot..."

场景2：简历上传页面
用户: 点击"上传简历" → 选择图片 → 预览识别结果 → 确认创建

场景3：JD 图片提取
用户: [上传 JD 截图]
AI: "我来帮你提取这个职位的技能要求..."
    [调用 OCR JD 识别工具]
AI: "提取成功！核心技能：Java、Spring Boot、MySQL、Redis..."
```

### 1.2 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 在聊天中上传图片附件 | P0 |
| OCR 识别 | 自动调用 OCR 识别图片内容 | P0 |
| 简历创建 | 基于识别结果创建简历记录 | P1 |
| JD 技能提取 | 从 JD 图片提取技能标签 | P1 |
| 识别结果展示 | 展示识别的文本内容 | P2 |

function JdSkillExtractor({ applicationId }: Props) {
  const [extracting, setExtracting] = useState(false);
  const [skills, setSkills] = useState<string[]>([]);

      {skills.length > 0 && (
        <SkillTagList
          skills={skills}
          editable
          onChange={setSkills}
        />
      )}
    </div>
  );
}

### 3.2 ImageDropZone 组件

```tsx
interface ImageDropZoneProps {
  accept: string;
  onUpload: (file: File) => void;
  ocrMode?: 'resume' | 'jd' | 'general';
  maxSize?: number;
  className?: string;
}

function ImageDropZone({
  accept,
  onUpload,
  ocrMode = 'general',
  maxSize = 10 * 1024 * 1024
}: ImageDropZoneProps) {
  const [dragging, setDragging] = useState(false);
  const [processing, setProcessing] = useState(false);

  const handleDrop = async (e: React.DragEvent) => {
    e.preventDefault();
    setDragging(false);

    const file = e.dataTransfer.files[0];
    if (!validateFile(file)) return;

    setProcessing(true);
    try {
      await onUpload(file);
    } finally {
      setProcessing(false);
    }
  };

  const validateFile = (file: File): boolean => {
    // 验证文件类型
    if (!accept.split(',').some(type => file.type.match(type))) {
      toast.error('不支持的文件类型');
      return false;
    }

    // 验证文件大小
    if (file.size > maxSize) {
      toast.error(`文件大小不能超过 ${maxSize / 1024 / 1024}MB`);
      return false;
    }

    return true;
  };

  return (
    <div
      className={cn(
        'image-drop-zone',
        dragging && 'dragging',
        processing && 'processing'
      )}
      onDragOver={(e) => { e.preventDefault(); setDragging(true); }}
      onDragLeave={() => setDragging(false)}
      onDrop={handleDrop}
    >
      <input
        type="file"
        accept={accept}
        onChange={(e) => {
          const file = e.target.files?.[0];
          if (file) handleDrop({ dataTransfer: { files: [file] } } as any);
        }}
        className="hidden-input"
      />

      {processing ? (
        <LoadingSpinner text="OCR 识别中..." />
      ) : (
        <div className="drop-content">
          <Upload className="icon" />
          <p>拖拽图片到此处</p>
          <p className="hint">或点击选择文件</p>
          <p className="formats">支持: JPG, PNG, PDF</p>
        </div>
      )}
    </div>
  );
}
```

### 3.3 OcrResultPreview 组件

```tsx
interface OcrResultPreviewProps {
  result: OcrResult;
  mode: 'resume' | 'jd' | 'general';
  onConfirm?: (data: any) => void;
}

function OcrResultPreview({ result, mode, onConfirm }: OcrResultPreviewProps) {
  if (result.status === 'pending') {
    return (
      <div className="ocr-preview ocr-preview--pending">
        <Spinner />
        <p>正在识别图片内容...</p>
      </div>
    );
  }

  if (result.status === 'failed') {
    return (
      <div className="ocr-preview ocr-preview--failed">
        <AlertCircle className="icon" />
        <p>识别失败</p>
        <p className="error">{result.error}</p>
        <Button variant="outline" size="sm">重新识别</Button>
      </div>
    );
  }

  return (
    <div className="ocr-preview ocr-preview--success">
      <div className="preview-header">
        <CheckCircle className="icon success" />
        <span>识别成功</span>
        <Badge variant="secondary">置信度: {(result.confidence! * 100).toFixed(0)}%</Badge>
      </div>

      {mode === 'resume' && (
        <ResumeDataPreview data={result.resumeData} />
      )}

      {mode === 'jd' && (
        <JdDataPreview data={result.jdData} />
      )}

      {mode === 'general' && (
        <TextPreview text={result.text} />
      )}

      {onConfirm && (
        <div className="preview-actions">
          <Button variant="outline" onClick={() => window.location.reload()}>
            重新上传
          </Button>
          <Button onClick={() => onConfirm(result.data)}>
            确认使用
          </Button>
        </div>
      )}
    </div>
  );
}
```

## 四、API 集成

### 4.1 OCR 相关 API

```typescript
// src/services/ocrApi.ts
export const ocrApi = {
  /**
   * 识别图片中的文字
   */
  async recognizeImage(imageData: string): Promise<OcrResult> {
    return request.post('/ocr/recognize', { imageData });
  },

  /**
   * 识别简历并解析结构化信息
   */
  async recognizeResume(file: File): Promise<ResumeInfo> {
    const formData = new FormData();
    formData.append('file', file);
    return request.post('/ocr/resume', formData, {
      headers: { 'Content-Type': 'multipart/form-data' }
    });
  },

  /**
   * 识别 JD 并提取技能
   */
  async recognizeJd(file: File, applicationId?: number): Promise<JdInfo> {
    const formData = new FormData();
    formData.append('file', file);
    if (applicationId) formData.append('applicationId', applicationId);
    return request.post('/ocr/jd', formData, {
      headers: { 'Content-Type': 'multipart/form-data' }
    });
  },

  /**
   * 获取 OCR 调用记录
   */
  async getRecords(page = 0, size = 20): Promise<OcrRecord[]> {
    return request.get('/ocr/records', { params: { page, size } });
  }
};
```

## 七、用户体验流程

### 7.1 聊天中发送图片

```
用户操作流程：
1. 点击输入框旁的图片图标
2. 选择或拍照上传图片
3. 预览图片，选择类型（简历/JD/普通）
4. 点击发送
5. 显示"识别中..."状态
6. AI 返回识别结果和回复

前端状态变化：
idle → selecting → previewing → sending → recognizing → completed
```

### 7.2 简历上传流程

```
用户操作流程：
1. 进入"上传简历"页面
2. 拖拽或选择简历图片
3. 系统自动识别（显示进度）
4. 预览识别结果（工作年限、技能等）
5. 编辑补充信息
6. 确认创建简历

错误处理：
- 识别失败 → 显示错误信息 + 重试按钮
- 置信度低 → 警告提示 + 建议手动录入
- 解析错误 → 显示原始文本 + 手动修正
```
