# 模拟面试会话创建时缺少快照数据修复

## 文档概述

**问题编号**: ISSUE-20260318-001
**发现日期**: 2026-03-18
**修复日期**: 2026-03-18
**严重程度**: 🔴 高（导致 Agent 无法获取必要上下文）

---

## 问题描述

### 问题现象

在创建模拟面试会话时，`MockInterviewSession` 的以下字段为 `null`：

- `resumeSnapshot` - 简历快照
- `jdSnapshot` - JD 要求快照
- `skillsSnapshot` - 技能标签快照

### 问题影响

1. **Agent 无法获取上下文**：后续的 Agent 评估任务（`evaluate()`, `decideNextStep()`, `askQuestion()`）都依赖这些快照数据
2. **评估结果不准确**：没有简历信息，评审专家无法对比简历声称与实际表现
3. **选题不精准**：副面试官无法根据 JD 要求选择合适的知识点
4. **问题不匹配**：主面试官无法根据候选人的技能水平调整问题难度

### 问题复现

```java
// 创建会话
MockInterviewSession session = interviewService.createSession(applicationId, resumeId, userId);

// 检查快照
System.out.println(session.getResumeSnapshot()); // null
System.out.println(session.getJdSnapshot());     // null

// 后续 Agent 调用
String context = buildReportContext(session, evaluations);
// context 中的简历快照和 JD 快照都是 null！
```

## 修复方案

### 1. 完善 `generateResumeSnapshot()` 方法

**功能**: 从简历和相关表中提取完整信息，生成 JSON 格式的快照

**包含信息**:
- ✅ 基本信息（工作年限、当前职位、目标级别）
- ✅ 自我介绍
- ✅ 技能列表（从 `resume_skills` 表查询）
  - skill_id
  - proficiency（熟练度）
  - experience_years（使用年限）
  - is_core（是否核心技能）
- ✅ 项目经验（从 `resume_projects` 表查询）
  - project_name
  - role
  - description
  - tech_stack

**实现代码**:

```java
public String generateResumeSnapshot(Long resumeId) {
    if (resumeId == null) {
        return null;
    }

    // 1. 查询简历基本信息
    UserResume resume = resumeMapper.selectById(resumeId);
    if (resume == null) {
        return null;
    }

    // 2. 查询简历技能
    List<ResumeSkill> skills = resumeSkillMapper.selectList(
        new LambdaQueryWrapper<ResumeSkill>()
            .eq(ResumeSkill::getResumeId, resumeId)
    );

    // 3. 查询项目经验
    List<ResumeProject> projects = resumeProjectMapper.selectList(
        new LambdaQueryWrapper<ResumeProject>()
            .eq(ResumeProject::getResumeId, resumeId)
    );

    // 4. 构建 JSON 快照
    StringBuilder snapshot = new StringBuilder();
    snapshot.append("{");
    snapshot.append("\"work_years\":").append(resume.getWorkYears()).append(",");
    snapshot.append("\"current_position\":\"").append(escapeJson(resume.getCurrentPosition())).append("\",");
    snapshot.append("\"target_level\":\"").append(escapeJson(resume.getTargetLevel())).append("\",");
    snapshot.append("\"summary\":\"").append(escapeJson(resume.getSummary())).append("\",");

    // 技能列表
    snapshot.append("\"skills\":[");
    for (ResumeSkill skill : skills) {
        snapshot.append("{");
        snapshot.append("\"skill_id\":").append(skill.getSkillId()).append(",");
        snapshot.append("\"proficiency\":\"").append(skill.getProficiencyLevel()).append("\",");
        snapshot.append("\"experience_years\":").append(skill.getExperienceYears()).append(",");
        snapshot.append("\"is_core\":").append(skill.getIsCoreSkill());
        snapshot.append("}");
    }
    snapshot.append("],");

    // 项目经验
    snapshot.append("\"projects\":[");
    for (ResumeProject project : projects) {
        snapshot.append("{");
        snapshot.append("\"project_name\":\"").append(project.getProjectName()).append("\",");
        snapshot.append("\"role\":\"").append(project.getRole()).append("\",");
        snapshot.append("\"description\":\"").append(project.getDescription()).append("\",");
        snapshot.append("\"tech_stack\":\"").append(project.getTechStack()).append("\"");
        snapshot.append("}");
    }
    snapshot.append("]");

    snapshot.append("}");
    return snapshot.toString();
}
```

### 2. 新增 `generateJdSnapshot()` 方法

**功能**: 从求职申请的 `skillsRequired` 字段生成 JSON 格式的快照

**包含信息**:
- ✅ 岗位信息（job_title, seniority_level, job_type）
- ✅ 岗位描述（job_description）
- ✅ 技能要求（skills_required）

**实现代码**:

```java
public String generateJdSnapshot(Long applicationId) {
    if (applicationId == null) {
        return null;
    }

    JobApplication application = applicationMapper.selectById(applicationId);
    if (application == null) {
        return null;
    }

    StringBuilder snapshot = new StringBuilder();
    snapshot.append("{");
    snapshot.append("\"job_title\":\"").append(escapeJson(application.getJobTitle())).append("\",");
    snapshot.append("\"seniority_level\":\"").append(escapeJson(application.getSeniorityLevel())).append("\",");
    snapshot.append("\"job_type\":\"").append(escapeJson(application.getJobType())).append("\",");
    snapshot.append("\"job_description\":\"").append(escapeJson(application.getJobDescription())).append("\",");

    // 技能要求
    snapshot.append("\"skills_required\":");
    String skillsRequired = application.getSkillsRequired();
    if (skillsRequired != null && skillsRequired.trim().startsWith("[")) {
        // 已经是 JSON 格式
        snapshot.append(skillsRequired);
    } else {
        // 逗号分隔的字符串，转换为 JSON 数组
        snapshot.append("[");
        String[] skills = skillsRequired.split(",");
        for (int i = 0; i < skills.length; i++) {
            snapshot.append("\"").append(escapeJson(skills[i].trim())).append("\"");
            if (i < skills.length - 1) {
                snapshot.append(",");
            }
        }
        snapshot.append("]");
    }

    snapshot.append("}");
    return snapshot.toString();
}
```

**输出示例**:

```json
{
  "job_title": "后端工程师",
  "seniority_level": "MIDDLE",
  "job_type": "全职",
  "job_description": "负责后端系统开发，要求精通Java、Spring Boot...",
  "skills_required": ["Java", "Spring Boot", "Redis", "MySQL", "Kafka"]
}
```

### 3. 修改 `createSession()` 方法

**关键改动**: 在创建会话时生成并设置快照

```java
@Transactional
public MockInterviewSession createSession(Long applicationId, Long resumeId, Long userId) {
    // 1. 获取申请信息
    JobApplication application = applicationMapper.selectById(applicationId);

    // 2. 确定简历ID
    resumeId = resumeId == null ? application.getResumeId() : resumeId;

    // 3. ⚠️ 生成简历快照（包含技能、项目等完整信息）
    String resumeSnapshot = generateResumeSnapshot(resumeId);
    if (resumeSnapshot == null) {
        log.warn("生成简历快照失败，使用空快照");
        resumeSnapshot = "{}";
    }

    // 4. ⚠️ 生成 JD 快照（包含技能要求等完整信息）
    String jdSnapshot = generateJdSnapshot(applicationId);
    if (jdSnapshot == null) {
        log.warn("生成 JD 快照失败，使用空快照");
        jdSnapshot = "{}";
    }

    // 5. 创建会话（包含快照）
    MockInterviewSession session = MockInterviewSession.builder()
            .sessionId(sessionId)
            .applicationId(applicationId)
            .userId(userId)
            .resumeId(resumeId)
            .companyId(application.getCompanyId())
            .jobTitle(application.getJobTitle())
            .seniorityLevel(application.getSeniorityLevel())
            .resumeSnapshot(resumeSnapshot)  // ⚠️ 设置简历快照
            .jdSnapshot(jdSnapshot)           // ⚠️ 设置 JD 快照
            .state(MockInterviewSession.InterviewState.INIT.name())
            .currentRound(0)
            .totalRounds(25)
            .createdAt(LocalDateTime.now())
            .build();

    sessionMapper.insert(session);

    log.info("创建面试会话成功，快照已生成");
    return session;
}
```

private String escapeJson(String value) {
    if (value == null) {
        return "";
    }
    return value
            .replace("\\", "\\\\")
            .replace("\"", "\\\"")
            .replace("\n", "\\n")
            .replace("\r", "\\r")
            .replace("\t", "\\t");
}

## 注意事项

### ⚠️ 重要提醒

1. **快照的不可变性**
   - 快照在创建会话时生成，之后不会更新
   - 即使用户修改了简历或 JD，已创建的会话快照保持不变
   - 这样保证了面试评估的历史准确性

2. **性能考虑**
   - 快照生成涉及多次数据库查询（简历、技能、项目）
   - 对于包含大量技能或项目的简历，可能耗时较长
   - 建议后续考虑异步生成或缓存优化

3. **JSON 转义**
   - 使用 `escapeJson()` 方法处理特殊字符
   - 防止 JSON 解析错误
   - 确保快照格式正确

4. **错误处理**
   - 如果快照生成失败，使用空 JSON `"{}"`
   - 记录警告日志，但不中断会话创建流程
   - 后续 Agent 需要处理空快照的情况

## 后续优化建议

### 1. 异步生成快照

**问题**: 快照生成可能耗时较长，影响会话创建响应时间

**方案**: 使用异步任务生成快照

```java
@Async
public CompletableFuture<String> generateResumeSnapshotAsync(Long resumeId) {
    return CompletableFuture.supplyAsync(() -> generateResumeSnapshot(resumeId));
}
```

```java
@Cacheable(value = "resumeSnapshots", key = "#resumeId")

    // ...

```

### 3. 快照版本控制

**问题**: 简历更新后，旧快照可能过时

### 4. 快照验证

**问题**: 快照可能格式错误或内容不完整

public boolean validateSnapshot(String snapshot) {
    try {
        JsonObject json = JsonParser.parseString(snapshot).getAsJsonObject();
        return json.has("work_years") && json.has("skills");

        return false;
