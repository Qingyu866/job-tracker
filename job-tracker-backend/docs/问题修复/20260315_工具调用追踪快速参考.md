# Tool Call Tracking 快速参考

## 🚀 快速开始

在工具方法中添加记录调用，只需 3 步：

### 1. 导入记录器

```java
import com.jobtracker.agent.tools.ToolCallTrace;
```

### 2. 在方法中添加记录

```java
public ToolResult myToolMethod(String param) {
    long startTime = System.currentTimeMillis();  // 开始计时

    try {
        // 你的业务逻辑
        ToolResult result = doSomething(param);

        // 记录成功
        ToolCallTrace.record("myToolMethod", param, result, startTime);
        return result;

    } catch (Exception e) {
        // 记录失败
        ToolCallTrace.recordFailure("myToolMethod", param, e, startTime);
        throw e;
    }
}
```

### 3. 验证

```sql
SELECT * FROM tool_call_records ORDER BY created_at DESC LIMIT 5;
```

## ⚠️ 常见问题

### Q: 为什么不用 AOP？
A: Spring AOP 的代理类会导致 LangChain4j 找不到 `@Tool` 注解。

### Q: 手动记录会不会很麻烦？
A: 是的，但这是唯一的解决方案。你可以逐步添加，先处理高频工具。

### Q: 记录会影响性能吗？
A: 影响很小，每次只增加 1 次数据库 INSERT 操作。

### Q: 如何清理历史数据？
A: 可以定期执行：
```sql
DELETE FROM tool_call_records WHERE created_at < DATE_SUB(NOW(), INTERVAL 30 DAY);
```
