**根本原因**：
- 新创建的服务使用了 `ChatLanguageModel`
- 但项目中实际使用的是 `OpenAiChatModel`

**当前错误代码**：

```java
import dev.langchain4j.model.openai.OpenAiChatModel;  // ✅ 使用项目中已有的

private final OpenAiChatModel chatModel;  // ✅ 正确类型
```

## 修复后的预期流程

### 1. 用户创建模拟面试会话

```
POST /api/mock-interview/start
{
  "applicationId": 4,
  "resumeId": 1
}
```

### 2. MockInterviewService.createSession() 执行流程

```
1. 读取申请和简历信息
   ↓
2. 调用 generateResumeSnapshot(resumeId)
   ├─ 从数据库读取 resume_skills、resume_projects 等表
   └─ 返回 JSON（如果表为空，返回空数组）
   ↓
3. 检查 isEmptySnapshot(resumeSnapshot)
   ├─ 如果 skills 为空数组 → 返回 true
   └─ 如果 projects 为空数组 → 返回 true
   ↓
4. 如果为空，调用 generateResumeSnapshotWithAI(resumeId)
   ├─ 构建简历文本（从已有数据）
   ├─ 调用 ResumeParsingService.parseResume()
   ├─ AI 解析返回结构化数据
   └─ 转换为 JSON
   ↓
5. 同样处理 JD 快照
   ↓
6. 创建会话，保存快照
```
