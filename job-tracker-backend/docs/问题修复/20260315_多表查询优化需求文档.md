# 多表查询优化需求文档

## 1. 文档概述

本文档记录了项目中需要多表查询但尚未完全实现的功能点，以及相应的修改方案。主要涉及 AI 工具返回值、REST API 响应数据以及统计功能的优化。

**版本**: 1.0.0
**创建日期**: 2026-03-15
**作者**: Job Tracker Team

## 2. 问题清单

### 2.1 面试查询工具 - N+1 查询问题

**文件位置**: `src/main/java/com/jobtracker/agent/tools/query/InterviewQueryTools.java`

**问题描述**:
在 `listInterviews`、`getUpcomingInterviews`、`getFollowUpRequiredInterviews` 等方法中，每个面试记录都需要单独查询关联的申请和公司信息，导致 N+1 查询问题。

**当前实现**:
```java
for (InterviewRecord interview : interviews) {
    // 每次循环都查询一次申请信息
    JobApplication app = applicationService.getById(interview.getApplicationId());
    // 再查询一次公司信息
    String companyJob = app != null ?
        toolHelper.getCompanyName(app.getCompanyId()) + " - " + app.getJobTitle() : "未知申请";
    // ...
}
```

**修改方案**:
1. 创建面试详情 DTO，包含关联信息
2. 在 Mapper 层实现多表关联查询
3. 或在 Service 层批量预加载关联数据

### 2.2 申请详情查询 - 缺少面试进度信息

**文件位置**: `src/main/java/com/jobtracker/agent/tools/query/ApplicationQueryTools.java`

**问题描述**:
`getApplicationById` 方法返回的申请详情只包含基本信息和公司信息，没有包含面试进度（InterviewProgress）和面试统计信息。

**当前返回内容**:
- 公司信息
- 职位信息
- 状态、类型、地点
- 薪资、优先级、日期
- 备注

**缺失内容**:
- 面试进度（第几轮、总共几轮）
- 面试统计（已通过/未通过/待进行）
- 最近面试时间

**修改方案**:
1. 在 `ApplicationQueryTools.getApplicationById` 中调用 `InterviewManagementService.getProgress`
2. 返回结果中增加面试进度字段

### 2.3 统计工具 - 状态统计不完整

**文件位置**:
- `src/main/java/com/jobtracker/agent/tools/statistics/StatisticsTools.java`
- `src/main/java/com/jobtracker/service/ExportService.java`

**问题描述**:
面试统计只计算 `COMPLETED` 状态，没有使用新的面试状态枚举（PASSED、FAILED、IN_PROGRESS）。

**当前实现**:
```java

long completed = allInterviews.stream()
    .filter(i -> ToolConstants.INTERVIEW_COMPLETED.equals(i.getStatus()))
    .count();
```

**修改方案**:
1. 更新统计逻辑，使用新的状态枚举
2. 增加通过率、失败率等统计指标
3. 增加当前轮次的统计

### 2.5 REST API 面试列表 - 缺少关联数据

**文件位置**: `src/main/java/com/jobtracker/controller/DataController.java`

**问题描述**:
面试相关的 REST API 返回的是原始实体，没有包含关联的申请和公司信息，前端需要额外请求。

**涉及的接口**:
- `GET /data/interviews` - 获取所有面试
- `GET /data/interviews/application/{applicationId}` - 获取申请的面试
- `GET /data/interviews/upcoming` - 获取即将进行的面试

**修改方案**:
1. 创建面试详情 DTO（InterviewDetailDTO）
2. 包含关联的申请和公司信息
3. 或在响应中增加可选的 `includeRelated` 参数

        "LEFT JOIN job_applications ja ON i.application_id = ja.id " +

### 3.4 更新 ExportService

**修改**: `src/main/java/com/jobtracker/service/ExportService.java`

```java
// 注入 InterviewManagementService
private final InterviewManagementService interviewManagementService;

public ApplicationDetailDTO getApplicationDetail(Long applicationId) {
    // ... 现有逻辑 ...

    // 增加面试进度
    InterviewProgress progress = interviewManagementService.getProgress(applicationId);

    return ApplicationDetailDTO.builder()
            .application(application)
            .company(company)
            .interviews(interviews)
            .logs(logs.stream().limit(20).collect(Collectors.toList()))
            .statistics(calculateStatistics(interviews))
            .interviewProgress(progress)  // 新增
            .build();
}

// 更新统计方法
private InterviewStatistics calculateStatistics(List<InterviewRecord> interviews) {
    long passed = interviews.stream()
        .filter(i -> "PASSED".equals(i.getStatus()))
        .count();

    long failed = interviews.stream()
        .filter(i -> "FAILED".equals(i.getStatus()))
        .count();

    long scheduled = interviews.stream()
        .filter(i -> "SCHEDULED".equals(i.getStatus()) || "IN_PROGRESS".equals(i.getStatus()))
        .count();

    long completed = passed + failed;

    double avgRating = interviews.stream()
        .filter(i -> i.getRating() != null &&
            ("PASSED".equals(i.getStatus()) || "FAILED".equals(i.getStatus()) || "COMPLETED".equals(i.getStatus())))
        .mapToInt(InterviewRecord::getRating)
        .average()
        .orElse(0.0);

    double passRate = completed > 0 ? (double) passed / completed * 100 : 0;

    return InterviewStatistics.builder()
        .total(interviews.size())
        .completed((int) completed)
        .scheduled((int) scheduled)
        .passed((int) passed)
        .failed((int) failed)
        .passRate(passRate)
        .averageRating(avgRating)
        .build();
}
```

### 3.7 优化 ToolHelper 批量查询

**修改**: `src/main/java/com/jobtracker/agent/tools/shared/ToolHelper.java`

```java
/**
 * 批量格式化面试列表（优化版）
 */
public String formatInterviewList(List<InterviewRecord> interviews) {
    if (interviews.isEmpty()) {
        return "";
    }

    // 批量预加载申请信息
    Set<Long> applicationIds = interviews.stream()
        .map(InterviewRecord::getApplicationId)
        .collect(Collectors.toSet());

    Map<Long, JobApplication> appMap = new HashMap<>();
    for (Long appId : applicationIds) {
        JobApplication app = applicationService.getById(appId);
        if (app != null) {
            appMap.put(appId, app);
        }
    }

    // 批量预加载公司信息
    Set<Long> companyIds = appMap.values().stream()
        .map(JobApplication::getCompanyId)
        .filter(Objects::nonNull)
        .collect(Collectors.toSet());

    Map<Long, Company> companyMap = new HashMap<>();
    for (Long companyId : companyIds) {
        Company company = companyService.getById(companyId);
        if (company != null) {
            companyMap.put(companyId, company);
        }
    }

    // 格式化输出
    StringBuilder sb = new StringBuilder();
    DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");

    for (int i = 0; i < interviews.size(); i++) {
        InterviewRecord interview = interviews.get(i);
        JobApplication app = appMap.get(interview.getApplicationId());
        Company company = app != null ? companyMap.get(app.getCompanyId()) : null;

        String companyJob = app != null ?
            (company != null ? company.getName() : "未知公司") + " - " + app.getJobTitle() : "未知申请";

        sb.append(String.format("%d. %s | %s面试 | %s\n",
            i + 1,
            companyJob,
            ToolConstants.getInterviewTypeDescription(interview.getInterviewType()),
            interview.getInterviewDate().format(formatter)
        ));
    }

    return sb.toString();
}
```
