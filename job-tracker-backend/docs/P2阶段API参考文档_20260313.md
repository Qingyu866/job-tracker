## 1. 聚合查询接口

### 1.1 获取申请详情（聚合信息）

**请求信息**：

| 项目 | 内容 |
|------|------|
| URL | `GET /api/data/applications/{id}/detail` |
| Content-Type | `application/json` |

**路径参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | Long | 是 | 申请ID |

## 2. 数据导出接口

### 2.1 导出Excel格式

**接口描述**：导出所有求职申请数据为Excel文件，使用EasyExcel组件

**请求信息**：

| 项目 | 内容 |
|------|------|
| URL | `GET /api/data/export/excel` |
| Content-Type | `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` |

**响应**：
- 返回可直接下载的Excel文件
- 文件名格式：`求职记录_yyyy-MM-dd.xlsx`

**Excel列定义**：

| 列名 | 宽度 | 说明 |
|------|------|------|
| ID | 10 | 申请ID |
| 职位名称 | 25 | 职位标题 |
| 公司名称 | 20 | 公司名称 |
| 状态 | 12 | 已翻译为中文 |
| 工作类型 | 12 | 全职/兼职/实习/合同 |
| 工作地点 | 15 | 工作地点 |
| 薪资范围 | 15 | 格式：20K-35K |
| 申请日期 | 12 | yyyy-MM-dd |
| 优先级 | 8 | 1-10 |
| 面试次数 | 10 | 关联的面试数量 |
| 备注 | 30 | 备注信息 |
| 创建时间 | 20 | yyyy-MM-dd HH:mm:ss |

**调用示例（浏览器）**：

```javascript
// 直接在浏览器中打开链接即可下载
window.open('http://localhost:8080/api/data/export/excel');
```

**调用示例（Axios）**：

```javascript
async function downloadExcel() {
  const response = await axios.get('/api/data/export/excel', {
    responseType: 'blob'
  });

  // 创建下载链接
  const url = window.URL.createObjectURL(new Blob([response.data]));
  const link = document.createElement('a');
  link.href = url;
  link.setAttribute('download', '求职记录.xlsx');
  document.body.appendChild(link);
  link.click();
  link.remove();
}
```

**响应示例**：

```json
  "message": "导出成功",
  "data": [

      "application": { ... },
      "company": { ... },
      "interviews": [ ... ],
      "logs": [ ... ],
      "statistics": { ... }

  ]

```

**调用示例**：

```bash
curl http://localhost:8080/api/data/export/json -o job_applications.json
```

```javascript
    type: 'application/json'
  });
```

## 3. 错误码说明

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 200 | 操作成功 | - |
| 400 | 请求参数错误 | 检查参数格式 |
| 404 | 资源不存在 | 确认ID是否正确 |
| 500 | 服务器内部错误 | 联系管理员或查看日志 |

## 4. 技术说明

### 4.1 EasyExcel 依赖

项目使用阿里巴巴开源的 EasyExcel 组件进行 Excel 导出：

```xml
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>easyexcel</artifactId>
    <version>3.3.4</version>
</dependency>
```

### 4.2 导出性能

- Excel导出：使用EasyExcel流式写入，支持大数据量导出
- JSON导出：聚合查询可能消耗较多内存，建议数据量超过1000条时使用分页导出

### 4.3 前端集成建议

```typescript
// 导出服务封装
export class ExportService {
  private API_BASE = '/api/data';

  // 下载Excel
  async downloadExcel(): Promise<void> {
    window.open(`${this.API_BASE}/export/excel`);
  }

  // 获取JSON数据
  async getJsonExport(): Promise<ApplicationDetailDTO[]> {
    const response = await axios.get(`${this.API_BASE}/export/json`);
    return response.data.data;
  }

  // 获取申请详情
  async getApplicationDetail(id: number): Promise<ApplicationDetailDTO> {
    const response = await axios.get(`${this.API_BASE}/applications/${id}/detail`);
    return response.data.data;
  }
}
```
