## 二、问题分析

### 2.1 🔴 严重问题

#### 问题1：工具描述不规范

**现状**：
```java
// 描述过于简单
@Tool("创建新的求职申请。参数：公司名称、职位名称...")

// 描述过于复杂，难以理解
@Tool("""
    【仅用于创建/添加新公司】当用户想要添加一个新公司到系统时使用。
    适用场景：用户明确说"创建"、"添加"、"新增"公司时才调用此工具。
    不要与查询混淆！...
    """)
```

**问题**：
- AI 难以准确理解工具用途
- 描述风格不统一
- 缺少结构化的参数说明

#### 问题2：参数命名不规范

**现状**：
```java
public String createApplication(
    String salaryMinStr,    // ❌ 带 Str 后缀
    String salaryMaxStr,    // ❌ 带 Str 后缀
    Integer priorityInt,    // ❌ 带 Int 后缀
    Integer durationMinutesInt  // ❌ 带 Int 后缀
)
```

**问题**：
- 命名冗余
- 类型转换逻辑散落在方法内部
- 不符合 Java 命名规范

#### 问题3：返回格式不统一

**现状**：
```java
// CompanyTools 返回 JSON
return objectMapper.writeValueAsString(companies);

// ApplicationTools 返回 Markdown 格式化文本
return "📋 找到 5 个求职申请：\n\n**职位名称**\n- 状态：...";
```

**问题**：
- AI 解析困难
- 前端无法复用
- 维护成本高

### 2.2 🔴 严重问题（续）

#### 问题4：参数设计不符合用户习惯

**现状**：工具要求用户提供ID
```java
@Tool("更新求职申请的状态。参数：申请ID、新状态")
public String updateApplicationStatus(Long applicationId, String newStatus)
```

**问题**：
- ❌ 用户不知道ID是什么
- ❌ 用户只会说"更新字节跳动的申请状态"
- ❌ AI 无法直接获取ID

**正确做法**：
```java
@Tool("更新求职申请状态。参数：公司名称或职位关键词、新状态")
public String updateApplicationStatus(String keyword, String newStatus)
```

**工作流程**：
1. AI 接收用户请求："把字节跳动的申请改为面试中"
2. AI 调用工具：`updateApplicationStatus("字节跳动", "INTERVIEW")`
3. 工具内部：通过关键词查找申请，找到后执行更新
4. 返回结果：告诉用户操作成功/失败

### 2.3 🟡 中等问题

#### 问题5：工具粒度不合理

**现状**：一个 `queryApplications` 方法支持4种不同筛选：
```java
if (keyword != null) { ... }
else if (status != null) { ... }
else if (priorityInt != null) { ... }
else if (days != null) { ... }
```

**问题**：
- AI 难以选择正确的筛选方式
- 参数组合复杂
- 测试困难

#### 问题6：缺少智能匹配

| 缺失功能 | 影响 |
|----------|------|
| 按名称/关键词更新 | 用户必须先查询再操作 |
| 按名称/关键词删除 | 用户体验差 |
| 模糊匹配 | 精确匹配容易失败 |
| 多结果处理 | 不知道选哪一个 |

### 2.4 🟢 轻微问题

- 错误消息格式不一致
- 缺少操作确认机制
- 工具间存在重复逻辑

## 三、改进方案

### 3.1 新的工具架构

```
agent/tools/
├── query/                      # 查询类工具
│   ├── ApplicationQueryTools.java
│   ├── CompanyQueryTools.java
│   └── InterviewQueryTools.java
├── command/                    # 命令类工具（创建/更新/删除）
│   ├── ApplicationCommandTools.java
│   ├── CompanyCommandTools.java
│   └── InterviewCommandTools.java
├── shared/                     # 共享组件
│   ├── ToolResult.java         # 统一返回格式
│   ├── ToolConstants.java      # 常量定义
│   └── ToolHelper.java         # 辅助方法
└── statistics/                 # 统计类工具
    └── StatisticsTools.java
```

### 3.2 核心设计原则：关键字优先（Keyword-First）

**原则**：工具参数设计应该以用户自然语言表达为核心，**用户永远不需要知道 ID**

| 操作类型 | 参数策略 | 示例 |
|----------|----------|------|
| **创建** | 名称（必填） | 公司名、职位名 |
| **查询** | 关键词/条件 | "字节跳动"、"面试中" |
| **更新** | 关键词 + 新值 | "字节跳动" + "INTERVIEW" |
| **删除** | 关键词（需确认） | "字节跳动的申请" |

**智能匹配策略**：

```
用户："把字节跳动的申请改为面试中"
      ↓
工具调用：updateApplicationStatus("字节跳动", "INTERVIEW")
      ↓
内部处理（ToolHelper.smartMatchApplication）：
  1. 搜索包含"字节跳动"的申请（公司名 + 职位名）
  2. 如果唯一匹配 → 直接更新，返回成功
  3. 如果多个匹配 → 返回列表，提示用户选择
     "找到多个申请：1.字节跳动-前端 2.字节跳动-后端，请告诉我具体是哪个？"
  4. 如果无匹配 → 返回错误，提供相似建议
```

**关键设计决策**：

1. **所有 Command 工具使用关键字参数**，不用 ID
2. **Query 工具保留 ID 参数**（AI 内部调用时使用）
3. **ToolHelper 提供统一的智能匹配逻辑**，处理单匹配/多匹配/无匹配场景

### 3.3 统一返回格式

```java
/**
 * 工具执行结果（统一格式）
 */
@Data
@Builder
public class ToolResult {
    /** 是否成功 */
    private boolean success;

    /** 消息类型：SUCCESS / ERROR / INFO */
    private String type;

    /** 消息内容（用户可见） */
    private String message;

    /** 详细数据（可选，AI可用） */
    private Object data;

    /** 错误代码（失败时） */
    private String errorCode;

    // ========== 静态工厂方法 ==========

    public static ToolResult success(String message) {
        return ToolResult.builder()
                .success(true)
                .type("SUCCESS")
                .message(message)
                .build();
    }

    public static ToolResult success(String message, Object data) {
        return ToolResult.builder()
                .success(true)
                .type("SUCCESS")
                .message(message)
                .data(data)
                .build();
    }

    public static ToolResult error(String message) {
        return ToolResult.builder()
                .success(false)
                .type("ERROR")
                .message(message)
                .build();
    }

    public static ToolResult error(String errorCode, String message) {
        return ToolResult.builder()
                .success(false)
                .type("ERROR")
                .errorCode(errorCode)
                .message(message)
                .build();
    }
}
```

## 四、详细设计

### 4.1 ApplicationQueryTools（查询工具）

```java
package com.jobtracker.agent.tools.query;

import com.jobtracker.agent.tools.shared.ToolResult;
import com.jobtracker.dto.ApplicationDetailDTO;
import com.jobtracker.entity.JobApplication;
import com.jobtracker.service.ApplicationService;
import dev.langchain4j.agent.tool.Tool;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.util.List;

@Slf4j
@Component
@RequiredArgsConstructor
public class ApplicationQueryTools {

    private final ApplicationService applicationService;

    /**
     * 根据ID获取申请详情
     */
    @Tool("""
        [查询] 获取求职申请详情

        适用场景：用户想查看某个具体申请的完整信息

        参数：
        - applicationId: 申请ID（必填，数字类型）

        返回：申请的详细信息，包括公司、面试记录等
        """)
    public ToolResult getApplicationById(Long applicationId) {
        log.info("AI调用：获取申请详情 id={}", applicationId);

        if (applicationId == null) {
            return ToolResult.error("PARAM_MISSING", "申请ID不能为空");
        }

        JobApplication app = applicationService.getById(applicationId);
        if (app == null) {
            return ToolResult.error("NOT_FOUND", "申请不存在，ID: " + applicationId);
        }

        return ToolResult.success("查询成功", app);
    }

    /**
     * 获取申请列表
     */
    @Tool("""
        [查询] 获取求职申请列表

        适用场景：
        - 用户想查看所有或部分申请
        - 按状态筛选申请
        - 搜索特定职位

        参数：
        - status: 按状态筛选（可选）WISHLIST/APPLIED/INTERVIEW/OFFER/REJECTED/WITHDRAWN
        - keyword: 按职位关键词搜索（可选）
        - limit: 返回数量限制（可选，默认10，最大50）

        返回：申请列表摘要
        """)
    public ToolResult listApplications(String status, String keyword, Integer limit) {
        log.info("AI调用：查询申请列表 status={}, keyword={}", status, keyword);

        limit = limit != null ? Math.min(limit, 50) : 10;

        List<JobApplication> applications;

        if (keyword != null && !keyword.isBlank()) {
            applications = applicationService.searchByJobTitle(keyword);
        } else if (status != null && !status.isBlank()) {
            applications = applicationService.listByStatus(status);
        } else {
            applications = applicationService.list();
        }

        // 限制返回数量
        if (applications.size() > limit) {
            applications = applications.subList(0, limit);
        }

        return ToolResult.success(
            String.format("找到 %d 个申请", applications.size()),
            applications
        );
    }
}
```

### 4.2 ApplicationCommandTools（命令工具 - 关键字优先）

```java
package com.jobtracker.agent.tools.command;

import com.jobtracker.agent.tools.shared.ToolHelper;
import com.jobtracker.agent.tools.shared.ToolResult;
import com.jobtracker.entity.Company;
import com.jobtracker.entity.JobApplication;
import com.jobtracker.service.ApplicationService;
import com.jobtracker.service.CompanyService;
import dev.langchain4j.agent.tool.Tool;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.math.BigDecimal;
import java.time.LocalDate;

@Slf4j
@Component
@RequiredArgsConstructor
public class ApplicationCommandTools {

    private final ApplicationService applicationService;
    private final CompanyService companyService;
    private final ToolHelper toolHelper;

    /**
     * 创建求职申请
     */
    @Tool("""
        [创建] 新建求职申请

        适用场景：用户想要记录一个新的求职申请

        必填参数：
        - companyName: 公司名称
        - jobTitle: 职位名称

        可选参数：
        - jobType: 工作类型 FULL_TIME/PART_TIME/CONTRACT/INTERNSHIP
        - workLocation: 工作地点
        - salaryMin: 薪资下限（数字）
        - salaryMax: 薪资上限（数字）
        - status: 初始状态 WISHLIST（默认）/APPLIED
        - priority: 优先级 1-10（默认5）
        - notes: 备注信息

        返回：创建结果，包含新申请ID
        """)
    public ToolResult createApplication(
            String companyName,
            String jobTitle,
            String jobType,
            String workLocation,
            BigDecimal salaryMin,
            BigDecimal salaryMax,
            String status,
            Integer priority,
            String notes
    ) {
        log.info("AI调用：创建申请 company={}, job={}", companyName, jobTitle);

        // 参数校验
        if (companyName == null || companyName.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "公司名称不能为空");
        }
        if (jobTitle == null || jobTitle.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "职位名称不能为空");
        }

        try {
            // 查找或创建公司
            Company company = companyService.getByName(companyName);
            if (company == null) {
                company = Company.builder().name(companyName).build();
                companyService.save(company);
            }

            // 创建申请
            JobApplication app = new JobApplication();
            app.setCompanyId(company.getId());
            app.setJobTitle(jobTitle);
            app.setJobType(jobType);
            app.setWorkLocation(workLocation);
            app.setSalaryMin(salaryMin);
            app.setSalaryMax(salaryMax);
            app.setStatus(status != null ? status : "WISHLIST");
            app.setPriority(priority != null ? priority : 5);
            app.setNotes(notes);

            if ("APPLIED".equals(app.getStatus())) {
                app.setApplicationDate(LocalDate.now());
            }

            applicationService.save(app);

            return ToolResult.success(
                String.format("✅ 成功创建申请：%s - %s（ID: %d）", companyName, jobTitle, app.getId()),
                app
            );
        } catch (Exception e) {
            log.error("创建申请失败", e);
            return ToolResult.error("CREATE_FAILED", "创建失败：" + e.getMessage());
        }
    }

    /**
     * 更新申请状态（关键字优先）
     */
    @Tool("""
        [更新] 通过关键词修改求职申请状态

        适用场景：
        - 用户说"把字节跳动的申请改为面试中"
        - 用户说"腾讯的申请拿到offer了"
        - 用户说"阿里巴巴那个不用跟了，撤回吧"

        参数：
        - keyword: 公司名称或职位关键词（必填）
          例如："字节跳动"、"前端工程师"、"字节-前端"
        - newStatus: 新状态（必填）
          WISHLIST/APPLIED/INTERVIEW/OFFER/REJECTED/WITHDRAWN

        返回：
        - 唯一匹配：直接更新，返回成功
        - 多个匹配：返回列表，提示用户选择
        - 无匹配：返回错误
        """)
    public ToolResult updateApplicationStatus(String keyword, String newStatus) {
        log.info("AI调用：更新申请状态 keyword={}, status={}", keyword, newStatus);

        if (keyword == null || keyword.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "请提供公司名称或职位关键词");
        }
        if (newStatus == null || newStatus.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "新状态不能为空");
        }

        try {
            // 智能匹配申请
            ToolResult matchResult = toolHelper.smartMatchApplication(keyword);

            if (!matchResult.isSuccess()) {
                return matchResult; // 无匹配，返回错误
            }

            // 检查是否需要用户选择
            if (matchResult.getData() instanceof java.util.List<?> list && list.size() > 1) {
                return ToolResult.builder()
                    .success(false)
                    .type("MULTIPLE_MATCH")
                    .message("找到多个申请，请告诉我具体是哪个？")
                    .data(list)
                    .build();
            }

            // 唯一匹配，执行更新
            JobApplication app = (JobApplication) matchResult.getData();
            applicationService.updateStatus(app.getId(), newStatus);

            return ToolResult.success(
                String.format("✅ 状态已更新：%s - %s → %s",
                    toolHelper.getCompanyName(app.getCompanyId()),
                    app.getJobTitle(),
                    newStatus)
            );
        } catch (Exception e) {
            log.error("更新状态失败", e);
            return ToolResult.error("UPDATE_FAILED", "更新失败：" + e.getMessage());
        }
    }

    /**
     * 更新申请详细信息（关键字优先）
     */
    @Tool("""
        [更新] 通过关键词修改求职申请详细信息

        适用场景：修改申请的备注、优先级等信息

        参数：
        - keyword: 公司名称或职位关键词（必填）
        - priority: 优先级 1-10（可选）
        - notes: 备注信息（可选）

        返回：更新结果
        """)
    public ToolResult updateApplicationDetails(
            String keyword,
            Integer priority,
            String notes
    ) {
        log.info("AI调用：更新申请详情 keyword={}", keyword);

        if (keyword == null || keyword.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "请提供公司名称或职位关键词");
        }

        try {
            ToolResult matchResult = toolHelper.smartMatchApplication(keyword);

            if (!matchResult.isSuccess()) {
                return matchResult;
            }

            if (matchResult.getData() instanceof java.util.List<?> list && list.size() > 1) {
                return ToolResult.builder()
                    .success(false)
                    .type("MULTIPLE_MATCH")
                    .message("找到多个申请，请告诉我具体是哪个？")
                    .data(list)
                    .build();
            }

            JobApplication app = (JobApplication) matchResult.getData();

            if (priority != null) {
                app.setPriority(priority);
            }
            if (notes != null) {
                app.setNotes(notes);
            }

            applicationService.updateById(app);

            return ToolResult.success(
                String.format("✅ 已更新：%s - %s",
                    toolHelper.getCompanyName(app.getCompanyId()),
                    app.getJobTitle())
            );
        } catch (Exception e) {
            log.error("更新申请失败", e);
            return ToolResult.error("UPDATE_FAILED", "更新失败：" + e.getMessage());
        }
    }

    /**
     * 删除申请（关键字优先）
     */
    @Tool("""

        参数：
        - keyword: 公司名称或职位关键词（必填）

        返回：删除结果或确认提示
        """)
    public ToolResult deleteApplication(String keyword) {
        log.info("AI调用：删除申请 keyword={}", keyword);

        if (keyword == null || keyword.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "请提供公司名称或职位关键词");
        }

        try {
            ToolResult matchResult = toolHelper.smartMatchApplication(keyword);

            if (!matchResult.isSuccess()) {
                return matchResult;
            }

            if (matchResult.getData() instanceof java.util.List<?> list && list.size() > 1) {
                return ToolResult.builder()
                    .success(false)
                    .type("MULTIPLE_MATCH")
                    .message("找到多个申请，请告诉我具体要删除哪个？")

                    .build();
            }
```

### 4.3 ToolHelper（智能匹配辅助类 - 核心组件）

```java
package com.jobtracker.agent.tools.shared;

import com.jobtracker.entity.Company;
import com.jobtracker.entity.InterviewRecord;
import com.jobtracker.entity.JobApplication;
import com.jobtracker.service.ApplicationService;
import com.jobtracker.service.CompanyService;
import com.jobtracker.service.InterviewService;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.time.LocalDate;
import java.time.LocalDateTime;
import java.util.ArrayList;
import java.util.List;
import java.util.stream.Collectors;

/**
 * 工具辅助类 - 提供智能匹配逻辑
 * <p>
 * 核心功能：
 * 1. 根据关键字智能匹配申请/面试/公司
 * 2. 处理单匹配/多匹配/无匹配场景
 * 3. 提供格式化输出
 * </p>
 */
@Slf4j
@Component
@RequiredArgsConstructor
public class ToolHelper {

    private final ApplicationService applicationService;
    private final CompanyService companyService;
    private final InterviewService interviewService;

    // ========== 申请匹配 ==========

    /**
     * 智能匹配申请
     * <p>
     * 搜索策略：
     * 1. 精确匹配公司名
     * 2. 模糊匹配公司名
     * 3. 匹配职位名
     * 4. 组合匹配（公司-职位）
     * </p>
     *
     * @param keyword 关键词（公司名、职位名、或组合）
     * @return 匹配结果
     */
    public ToolResult smartMatchApplication(String keyword) {
        log.debug("智能匹配申请: keyword={}", keyword);

        List<JobApplication> matches = new ArrayList<>();

        // 1. 尝试精确匹配公司名
        Company company = companyService.getByName(keyword);
        if (company != null) {
            matches.addAll(applicationService.listByCompanyId(company.getId()));
        }

        // 2. 模糊搜索公司名
        if (matches.isEmpty()) {
            List<Company> companies = companyService.searchByName(keyword);
            for (Company c : companies) {
                matches.addAll(applicationService.listByCompanyId(c.getId()));
            }
        }

        // 3. 搜索职位名
        if (matches.isEmpty()) {
            matches.addAll(applicationService.searchByJobTitle(keyword));
        }

        // 4. 组合搜索（如 "字节-前端"）
        if (matches.isEmpty() && keyword.contains("-")) {
            String[] parts = keyword.split("-", 2);
            String companyPart = parts[0].trim();
            String jobPart = parts[1].trim();

            List<Company> companies = companyService.searchByName(companyPart);
            for (Company c : companies) {
                List<JobApplication> apps = applicationService.listByCompanyId(c.getId());
                for (JobApplication app : apps) {
                    if (app.getJobTitle() != null && app.getJobTitle().contains(jobPart)) {
                        matches.add(app);
                    }
                }
            }
        }

        // 处理结果
        return handleMatchResult(matches, "申请", keyword);
    }

    // ========== 面试匹配 ==========

    /**
     * 智能匹配面试
     * <p>
     * 搜索策略：
     * 1. 通过公司名/职位名找到申请，再找面试
     * 2. 支持时间关键词（今天、明天、本周）
     * </p>
     *
     * @param keyword 关键词（公司名、职位名、或时间）
     * @return 匹配结果
     */
    public ToolResult smartMatchInterview(String keyword) {
        log.debug("智能匹配面试: keyword={}", keyword);

        List<InterviewRecord> matches = new ArrayList<>();

        // 1. 尝试时间关键词
        if (isTimeKeyword(keyword)) {
            matches.addAll(searchInterviewsByTimeKeyword(keyword));
        }

        // 2. 通过申请匹配
        if (matches.isEmpty()) {
            ToolResult appResult = smartMatchApplication(keyword);
            if (appResult.isSuccess() && appResult.getData() != null) {
                if (appResult.getData() instanceof JobApplication app) {
                    matches.addAll(interviewService.listByApplicationId(app.getId()));
                } else if (appResult.getData() instanceof List<?> list) {
                    for (Object obj : list) {
                        if (obj instanceof JobApplication appItem) {
                            matches.addAll(interviewService.listByApplicationId(appItem.getId()));
                        }
                    }
                }
            }
        }

        return handleMatchResult(matches, "面试", keyword);
    }

    // ========== 公司匹配 ==========

    /**
     * 智能匹配公司
     */
    public ToolResult smartMatchCompany(String keyword) {
        log.debug("智能匹配公司: keyword={}", keyword);

        List<Company> matches = new ArrayList<>();

        // 1. 精确匹配
        Company exact = companyService.getByName(keyword);
        if (exact != null) {
            matches.add(exact);
        }

        // 2. 模糊匹配
        if (matches.isEmpty()) {
            matches.addAll(companyService.searchByName(keyword));
        }

        return handleMatchResult(matches, "公司", keyword);
    }

    // ========== 辅助方法 ==========

    /**
     * 处理匹配结果
     */
    private <T> ToolResult handleMatchResult(List<T> matches, String entityType, String keyword) {
        if (matches.isEmpty()) {
            return ToolResult.error(
                "NOT_FOUND",
                String.format("未找到%s：\"%s\"", entityType, keyword)
            );
        }

        if (matches.size() == 1) {
            return ToolResult.success("找到唯一匹配", matches.get(0));
        }

        // 多个匹配，返回列表供选择
        return ToolResult.builder()
            .success(true)
            .type("MULTIPLE_MATCH")
            .message(String.format("找到 %d 个%s，请选择：", matches.size(), entityType))
            .data(matches)
            .build();
    }

    /**
     * 判断是否为时间关键词
     */
    private boolean isTimeKeyword(String keyword) {
        if (keyword == null) return false;
        String lower = keyword.toLowerCase();
        return lower.contains("今天") || lower.contains("明天") ||
               lower.contains("后天") || lower.contains("昨天") ||
               lower.contains("本周") || lower.contains("下周") ||
               lower.contains("最近");
    }

    /**
     * 根据时间关键词搜索面试
     */
    private List<InterviewRecord> searchInterviewsByTimeKeyword(String keyword) {
        String lower = keyword.toLowerCase();
        LocalDate today = LocalDate.now();

        if (lower.contains("今天")) {
            return interviewService.listByDate(today);
        } else if (lower.contains("明天")) {
            return interviewService.listByDate(today.plusDays(1));
        } else if (lower.contains("后天")) {
            return interviewService.listByDate(today.plusDays(2));
        } else if (lower.contains("本周")) {
            LocalDate weekEnd = today.plusDays(7 - today.getDayOfWeek().getValue());
            return interviewService.listByDateRange(today, weekEnd);
        } else if (lower.contains("最近")) {
            return interviewService.listRecent(7);
        }

        return new ArrayList<>();
    }

    /**
     * 获取公司名称（辅助方法）
     */
    public String getCompanyName(Long companyId) {
        if (companyId == null) return "未知公司";
        Company company = companyService.getById(companyId);
        return company != null ? company.getName() : "未知公司";
    }

    /**
     * 格式化申请列表（用于多匹配提示）
     */
    public String formatApplicationList(List<JobApplication> applications) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < applications.size(); i++) {
            JobApplication app = applications.get(i);
            sb.append(String.format("%d. %s - %s (状态: %s)\n",
                i + 1,
                getCompanyName(app.getCompanyId()),
                app.getJobTitle(),
                app.getStatus()
            ));
        }
        return sb.toString();
    }
}
```

```java
        适用场景：
        - 用户说"帮我给字节跳动的申请添加一个面试"

        - interviewDate: 面试时间（必填）

          如果只传日期，默认 09:00
        - interviewerName: 面试官姓名（可选）
        - interviewerTitle: 面试官职位（可选）
        - durationMinutes: 面试时长（分钟，可选）

        返回：创建结果
        """)
    public ToolResult createInterview(

            String interviewType,
            String interviewDate,
            String interviewerName,
            String interviewerTitle,
            Integer durationMinutes
    ) {
        log.info("AI调用：创建面试 keyword={}, type={}, date={}", keyword, interviewType, interviewDate);

        if (keyword == null || keyword.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "请提供公司名称或职位关键词");
        }
        if (interviewDate == null || interviewDate.isBlank()) {
            return ToolResult.error("PARAM_MISSING", "面试时间不能为空");

            // 智能匹配申请
            ToolResult matchResult = toolHelper.smartMatchApplication(keyword);

            if (!matchResult.isSuccess()) {
                return ToolResult.error("NOT_FOUND",
                    String.format("未找到申请：\"%s\"，请先创建申请", keyword));

            if (matchResult.getData() instanceof java.util.List<?> list && list.size() > 1) {

                    .success(false)
                    .type("MULTIPLE_MATCH")
                    .message("找到多个申请，请告诉我具体是哪个？")
                    .data(list)
                    .build();
            }

                    record.getInterviewDate().format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm")))
            );

            log.error("创建面试失败", e);
            return ToolResult.error("CREATE_FAILED", "创建失败：" + e.getMessage());
```
