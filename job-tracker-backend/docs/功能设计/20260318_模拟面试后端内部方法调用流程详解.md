**AgentFactory.createAgents() 内部流程**:
```java
public InterviewAgents createAgents(MockInterviewSession session) {
    String sessionId = session.getSessionId();

    // 3.2.1 检查缓存
    if (agentsCache.containsKey(sessionId)) {
        return agentsCache.get(sessionId);
    }

    // 3.2.2 为每个 Agent 创建独立的 ChatMemory
    ChatMemory mainMemory = memoryProvider.getChatMemory(
        sessionId, AgentType.MAIN_INTERVIEWER
    );
    ChatMemory viceMemory = memoryProvider.getChatMemory(
        sessionId, AgentType.VICE_INTERVIEWER
    );
    ChatMemory evalMemory = memoryProvider.getChatMemory(
        sessionId, AgentType.EVALUATOR
    );

    // 3.2.3 构建上下文
    String context = buildContext(session);

    // 3.2.4 创建主面试官
    MainInterviewerAgent mainInterviewer = AiServices.builder(MainInterviewerAgent.class)
        .chatModel(chatModel)
        .chatMemory(mainMemory)
        .systemMessage(buildMainInterviewerSystemMessage(context))
        .build();

    // 3.2.5 创建副面试官
    ViceInterviewerAgent viceInterviewer = AiServices.builder(ViceInterviewerAgent.class)
        .chatModel(chatModel)
        .chatMemory(viceMemory)
        .systemMessage(buildViceInterviewerSystemMessage(context))
        .build();

    // 3.2.6 创建评审专家
    ExpertEvaluatorAgent evaluator = AiServices.builder(ExpertEvaluatorAgent.class)
        .chatModel(chatModel)
        .chatMemory(evalMemory)
        .systemMessage(buildExpertEvaluatorSystemMessage(context))
        .build();

    // 3.2.7 组装并缓存
    InterviewAgents agents = new InterviewAgents(
        sessionId, mainInterviewer, viceInterviewer, evaluator
    );
    agentsCache.put(sessionId, agents);

    return agents;
}
```

**内存操作**:
- `ConcurrentHashMap<String, InterviewAgents>` 缓存
- 每个Agent有独立的 `ChatMemory` 实例

**返回值**: `InterviewAgents` 对象，包含三个Agent

public InterviewMessage getLastQuestion(String sessionId) {
    List<InterviewMessage> messages = messageMapper.selectList(
        new LambdaQueryWrapper<InterviewMessage>()
            .eq(InterviewMessage::getSessionId, sessionId)
            .eq(InterviewMessage::getRole, InterviewMessage.MessageRole.ASSISTANT.name())

    return messages.isEmpty() ? null : messages.get(0);
}

private MockInterviewEvaluation parseAndCreateEvaluation(
    String sessionId,
    Integer roundNumber,
    String question,
    String userAnswer,
    String evaluationResult
) {
    try {
        // 7.1.1 去除 Markdown 代码块标记
        String cleanedJson = evaluationResult
            .replaceAll("```json\\s*", "")
            .replaceAll("```\\s*$", "")
            .trim();

        // 7.1.2 配置宽松解析模式
        ObjectMapper mapper = new ObjectMapper();
        mapper.configure(JsonParser.Feature.ALLOW_UNQUOTED_CONTROL_CHARS, true);
        mapper.configure(JsonParser.Feature.ALLOW_BACKSLASH_ESCAPING_ANY_CHARACTER, true);

        // 7.1.3 解析 JSON
        Map<String, Object> result = mapper.readValue(cleanedJson, Map.class);

        // 7.1.4 提取评分
        Map<String, Object> scores = (Map<String, Object>) result.get("scores");
        Double technical = scores != null ? ((Number) scores.get("technical")).doubleValue() : 0.0;
        Double logic = scores != null ? ((Number) scores.get("logic")).doubleValue() : 0.0;
        Double depth = scores != null ? ((Number) scores.get("depth")).doubleValue() : 0.0;
        Double total = result.get("total_score") != null ?
            ((Number) result.get("total_score")).doubleValue() :
            (technical + logic + depth);

        // 7.1.5 提取反馈
        String feedback = (String) result.get("feedback");
        String suggestion = (String) result.get("suggestion");

        // 7.1.6 创建评分记录
        MockInterviewEvaluation evaluation = MockInterviewEvaluation.builder()
            .sessionId(sessionId)
            .roundNumber(roundNumber)
            .skillId(0L)  // 0 表示通用技能
            .skillName("通用")
            .questionText(question)
            .userAnswer(userAnswer)
            .technicalScore(BigDecimal.valueOf(technical))
            .logicScore(BigDecimal.valueOf(logic))
            .depthScore(BigDecimal.valueOf(depth))
            .totalScore(BigDecimal.valueOf(total))
            .feedback(feedback)
            .suggestion(suggestion)
            .build();

        // 7.1.7 保存到数据库
        return evaluationService.createEvaluation(evaluation);

    } catch (Exception e) {
        // 7.1.8 解析失败，使用默认值
        MockInterviewEvaluation evaluation = MockInterviewEvaluation.builder()
            .sessionId(sessionId)
            .roundNumber(roundNumber)
            .skillId(0L)
            .skillName("通用")
            .questionText(question)
            .userAnswer(userAnswer)
            .technicalScore(BigDecimal.valueOf(2.5))
            .logicScore(BigDecimal.valueOf(2.5))
            .depthScore(BigDecimal.valueOf(2.5))
            .totalScore(BigDecimal.valueOf(7.5))
            .feedback(evaluationResult)
            .build();

        return evaluationService.createEvaluation(evaluation);
    }
}
```

**persistMemory() 内部流程**:
```java
private void persistMemory(String sessionId, AgentType agentType) {
    // 8.1.1.1 获取 ChatMemory
    ChatMemory memory = getChatMemory(sessionId, agentType);

    // 8.1.1.2 转换为 JSON
    SessionMemories sessionMemories = new SessionMemories();
    sessionMemories.setSessionId(sessionId);
    sessionMemories.setAgentType(agentType.name());

    List<MessageData> messages = new ArrayList<>();
    for (ChatMessage msg : memory.messages()) {
        MessageData data = new MessageData();
        data.setType(msg.getClass().getSimpleName());
        data.setText(msg.text());
        messages.add(data);
    }
    sessionMemories.setMessages(messages);

    // 8.1.1.3 查询是否已存在
    LambdaQueryWrapper<InterviewMemory> wrapper = new LambdaQueryWrapper<InterviewMemory>()
        .eq(InterviewMemory::getSessionId, sessionId)
        .eq(InterviewMemory::getAgentType, agentType.name());

    InterviewMemory existing = memoryMapper.selectOne(wrapper);

    if (existing != null) {
        // 8.1.1.4 更新现有记录
        existing.setMemoryData(JSON.toJSONString(sessionMemories));
        existing.setUpdatedAt(LocalDateTime.now());
        memoryMapper.updateById(existing);
    } else {
        // 8.1.1.5 插入新记录
        InterviewMemory newMemory = InterviewMemory.builder()
            .sessionId(sessionId)
            .agentType(agentType.name())
            .memoryData(JSON.toJSONString(sessionMemories))
            .createdAt(LocalDateTime.now())
            .updatedAt(LocalDateTime.now())
            .build();
        memoryMapper.insert(newMemory);
    }
}
```

private String buildNextStepContext(MockInterviewSession session, String userAnswer) {
    // 9.1.1 获取最近对话历史
    List<InterviewMessage> recentMessages = messageService.getSessionMessages(session.getSessionId());

    // 9.1.2 提取最近 5 轮 AI 回复
    String recentTopics = "";
    if (!recentMessages.isEmpty()) {
        StringBuilder topics = new StringBuilder();
        int count = 0;
        for (int i = recentMessages.size() - 1; i >= 0 && count < 5; i--) {
            InterviewMessage msg = recentMessages.get(i);
            if ("ASSISTANT".equals(msg.getRole()) && msg.getContent() != null) {
                topics.append(msg.getContent()).append("\n");
                count++;
            }
        }
        recentTopics = topics.toString();
    }

    // 9.1.3 构建上下文
    return String.format("""
        用户最新回答: %s
        已考察知识点: %s
        待考察知识点: %s

        # 最近 5 轮对话历史
        %s

        # 统计信息
        - 当前轮次: %d
        - 总消息数: %d
        """,
        userAnswer,
        session.getSkillsCovered() != null ? session.getSkillsCovered() : "无",
        session.getSkillsPending() != null ? session.getSkillsPending() : "无",
        recentTopics,
        session.getCurrentRound(),
        recentMessages.size()
    );
}
```

```java
public BigDecimal calculateTotalScore(String sessionId) {
    List<MockInterviewEvaluation> evaluations = getSessionEvaluations(sessionId);

    if (evaluations == null || evaluations.isEmpty()) {
        return BigDecimal.ZERO;
    }

    BigDecimal sum = BigDecimal.ZERO;
    for (MockInterviewEvaluation eval : evaluations) {
        if (eval.getTotalScore() != null) {
            sum = sum.add(eval.getTotalScore());
        }
    }

    // 计算平均分
    return sum.divide(
        BigDecimal.valueOf(evaluations.size()),
        2,
        RoundingMode.HALF_UP
    );
}
```

ORDER BY round_number ASC;

**计算逻辑**:
```
总分 = SUM(所有 evaluation 的 total_score) / evaluation 数量
```

- 持久化失败不影响提问流程
- 日志记录警告信息

## 五、性能分析

### 5.1 关键性能指标

| 指标 | sendMessage | finishInterview |
|-----|-------------|-----------------|
| 数据库操作次数 | 10次 | 9次 |
| Agent调用次数 | 3次 (评估+决策+提问) | 4次 (可信度分析+评分+建议+总结) |
| 预估响应时间 | 2-5秒 | 5-15秒 |
| 内存占用 | ~1MB | ~5MB |

### 5.2 性能瓶颈

1. **Agent 调用**: 每次调用需要等待 LLM 响应 (1-3秒)
2. **记忆持久化**: 每轮3次数据库写入
3. **上下文构建**: 获取历史消息需要查询数据库

### 5.3 优化建议

1. **批量持久化**: 将多个 Agent 记忆合并为一次数据库操作
2. **异步持久化**: 使用消息队列异步持久化记忆
3. **缓存优化**: 缓存历史消息，减少数据库查询
4. **并行调用**: 部分Agent调用可以并行执行

## 七、总结

本文档详细描述了模拟面试系统后端的两个核心流程：

1. **sendMessage**: 用户回答 → 评估 → 决策 → 提问
2. **finishInterview**: 生成完整报告（可信度分析+评分+建议+总结）

每个流程都包含了：
- 完整的方法调用链
- 数据库操作详情
- Agent 调用过程
- 异常处理策略
- 性能分析

**关键设计要点**:
- 三Agent架构，记忆完全隔离
- 每轮对话后持久化记忆，防止数据丢失
- 宽松JSON解析，处理AI返回的不规范格式
- 完善的异常处理和降级方案
- 数据库事务保证数据一致性
