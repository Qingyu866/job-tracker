# 状态流程设计文档

## 1. 当前状态分析

### 1.1 申请状态 (ApplicationStatus)

| 状态 | 代码 | 描述 | 问题 |
|------|------|------|------|
| 意愿清单 | WISHLIST | 尚未申请 | ✅ 正常 |
| 已申请 | APPLIED | 已提交申请 | ⚠️ 缺少"简历筛选"阶段 |
| 面试中 | INTERVIEW | 正在面试 | ⚠️ 太笼统，无法区分多轮面试 |
| 已录用 | OFFER | 收到Offer | ✅ 正常 |
| 已拒绝 | REJECTED | 被拒绝 | ✅ 正常 |
| 已撤回 | WITHDRAWN | 主动撤回 | ✅ 正常 |

### 1.2 面试状态 (InterviewStatus)

| 状态 | 代码 | 描述 | 问题 |
|------|------|------|------|
| 已安排 | SCHEDULED | 已预约 | ✅ 正常 |
| 已完成 | COMPLETED | 面试结束 | ⚠️ 缺少"通过/不通过"状态 |
| 已取消 | CANCELLED | 取消面试 | ✅ 正常 |
| 未参加 | NO_SHOW | 没去面试 | ✅ 正常 |

### 1.3 核心问题

1. **状态粒度不够**：APPLIED → INTERVIEW 跨度太大，缺少中间状态
2. **状态联动缺失**：面试状态变更不会自动更新申请状态
3. **无法区分结果**：面试 COMPLETED 后不知道是通过还是未通过
4. **缺少终态区分**：OFFER 后缺少"已接受/已拒绝"状态

---

## 2. 推荐方案对比

### 方案 A：保持现有状态 + 增强联动（最小改动）

**优点**：改动小，兼容现有数据
**缺点**：状态粒度仍然不够

### 方案 B：扩展状态枚举 + 自动联动（推荐）

**优点**：状态更清晰，自动化程度高
**缺点**：需要数据迁移

### 方案 C：引入状态机框架（过度设计）

**优点**：最规范
**缺点**：复杂度高，不适合当前规模

## 3. 推荐方案 B：详细设计

### 3.1 新的申请状态

```
WISHLIST      → 意愿清单（尚未申请）
    ↓
APPLIED       → 已申请（等待回复）
    ↓
SCREENING     → 简历筛选中（新增）
    ↓
INTERVIEW     → 面试中
    ↓
FINAL_ROUND   → 终面中（新增，可选）
    ↓
OFFERED       → 已收到Offer（从OFFER改名）

    └→ EXPIRED    → Offer已过期（新增）

任意状态 → REJECTED  → 被拒绝
任意状态 → WITHDRAWN → 已撤回
```

```
    ↓

    ↓

    └→ FAILED     → 未通过（新增）
```


### 3.6 面试与申请状态联动（多轮面试版）

```java
/**
 * 面试状态变更时的联动规则（支持多轮面试）
 */
public void onInterviewStatusChange(Long interviewId, InterviewStatus newStatus) {
    InterviewRecord interview = interviewMapper.selectById(interviewId);
    Long applicationId = interview.getApplicationId();

    switch (newStatus) {
        case SCHEDULED -> {
            // 安排面试 → 检查是否需要更新申请状态
            JobApplication app = applicationMapper.selectById(applicationId);
            // 只有在 INTERVIEW 之前的状态才更新
            if (isBeforeInterviewStage(app.getStatus())) {
                updateApplicationStatus(applicationId, ApplicationStatus.INTERVIEW);
            }
        }

        case PASSED -> {
            // 面试通过 → 检查是否还有后续面试
            boolean hasPendingInterviews = hasPendingOrScheduledInterviews(
                applicationId, interviewId);

            if (!hasPendingInterviews) {
                // 所有面试都通过了 → 等待最终结果
                log.info("所有面试已完成，等待HR反馈: applicationId={}", applicationId);
                // 状态保持 INTERVIEW，等待 Offer 或其他反馈
            }
        }

        case FAILED -> {
            // 面试未通过 → 判断流程是否结束
            if (interview.getIsFinal()) {
                // 终面未通过 → 直接拒绝
                updateApplicationStatus(applicationId, ApplicationStatus.REJECTED);
            } else {
                // 非终面未通过 → 取决于公司政策
                // 有些公司会给第二次机会，有些直接拒绝
                if (shouldRejectOnFail(applicationId)) {
                    updateApplicationStatus(applicationId, ApplicationStatus.REJECTED);
                } else {
                    // 取消后续面试安排
                    cancelPendingInterviews(applicationId);
                }
            }
        }

        case NO_SHOW -> {
            // 未参加 → 记录但不自动改变申请状态
            // 可以重新安排
        }
    }
}

/**
 * 检查是否有待处理/已安排的面试
 */
private boolean hasPendingOrScheduledInterviews(Long applicationId, Long excludeInterviewId) {
    return interviewMapper.countByApplicationAndStatus(
        applicationId,
        List.of(InterviewStatus.SCHEDULED, InterviewStatus.IN_PROGRESS),
        excludeInterviewId
    ) > 0;
}
```

### 3.8 面试进度计算

```java
/**
 * 计算面试进度
 */
public class InterviewProgress {
    private int totalRounds;      // 总轮次
    private int completedRounds;  // 已完成轮次
    private int passedRounds;     // 通过轮次
    private int currentRound;     // 当前进行中的轮次
    private String progressText;  // 进度文本描述

    public static InterviewProgress calculate(Long applicationId) {
        List<InterviewRecord> interviews = interviewMapper.findByApplicationId(applicationId);

        InterviewProgress progress = new InterviewProgress();
        progress.totalRounds = interviews.size();
        progress.completedRounds = (int) interviews.stream()
            .filter(i -> i.getStatus().isTerminal())
            .count();
        progress.passedRounds = (int) interviews.stream()
            .filter(i -> i.getStatus() == InterviewStatus.PASSED)
            .count();

        // 找到当前进行中的轮次
        interviews.stream()
            .filter(i -> i.getStatus() == InterviewStatus.SCHEDULED
                      || i.getStatus() == InterviewStatus.IN_PROGRESS)
            .findFirst()
            .ifPresent(i -> progress.currentRound = i.getRoundNumber());

        // 生成进度描述
        progress.progressText = generateProgressText(progress);

        return progress;
    }

    private static String generateProgressText(InterviewProgress p) {
        if (p.totalRounds == 0) return "暂无面试安排";
        if (p.passedRounds == p.totalRounds) return "全部面试通过，等待结果";
        if (p.currentRound > 0) return String.format("第%d轮面试进行中", p.currentRound);
        return String.format("面试进度: %d/%d", p.completedRounds, p.totalRounds);
    }
}
```

```java
package com.jobtracker.constants;

public enum ApplicationStatus {
    // 初始阶段
    WISHLIST("WISHLIST", "意愿清单", Stage.INITIAL),
    APPLIED("APPLIED", "已申请", Stage.APPLIED),
    SCREENING("SCREENING", "简历筛选中", Stage.IN_PROGRESS),

    // 面试阶段
    INTERVIEW("INTERVIEW", "面试中", Stage.IN_PROGRESS),
    FINAL_ROUND("FINAL_ROUND", "终面中", Stage.IN_PROGRESS),

    // Offer阶段
    OFFERED("OFFERED", "已收到Offer", Stage.OFFER),
    ACCEPTED("ACCEPTED", "已接受Offer", Stage.COMPLETED),
    DECLINED("DECLINED", "已拒绝Offer", Stage.COMPLETED),
    EXPIRED("EXPIRED", "Offer已过期", Stage.COMPLETED),

    // 终态
    REJECTED("REJECTED", "已拒绝", Stage.COMPLETED),
    WITHDRAWN("WITHDRAWN", "已撤回", Stage.COMPLETED);

    private final String code;
    private final String description;
    private final Stage stage;

    // ... 构造函数、getter 方法

    /**
     * 阶段分组
     */
    public enum Stage {
        INITIAL,      // 初始阶段
        APPLIED,      // 已申请
        IN_PROGRESS,  // 进行中
        OFFER,        // Offer阶段
        COMPLETED     // 已完成
    }

    /**
     * 是否为终态
     */
    public boolean isTerminal() {
        return stage == Stage.COMPLETED;
    }

    /**
     * 是否可以转换到目标状态
     */
    public boolean canTransitionTo(ApplicationStatus target) {
        return TransitionRules.APPLICATION_TRANSITIONS
            .getOrDefault(this, Collections.emptySet())
            .contains(target);
    }
}
```

```sql
-- 添加面试轮次相关字段
ALTER TABLE interview_records
ADD COLUMN round_number INT DEFAULT 1 COMMENT '面试轮次' AFTER interview_type,
ADD COLUMN is_final BOOLEAN DEFAULT FALSE COMMENT '是否为终面' AFTER round_number;

-- 添加申请进度视图（可选）
CREATE VIEW v_application_interview_progress AS
SELECT
    a.id AS application_id,
    a.status AS application_status,
    COUNT(i.id) AS total_interviews,
    SUM(CASE WHEN i.status = 'PASSED' THEN 1 ELSE 0 END) AS passed_interviews,
    SUM(CASE WHEN i.status = 'FAILED' THEN 1 ELSE 0 END) AS failed_interviews,
    SUM(CASE WHEN i.status IN ('SCHEDULED', 'IN_PROGRESS') THEN 1 ELSE 0 END) AS pending_interviews,
    MIN(CASE WHEN i.status IN ('SCHEDULED', 'IN_PROGRESS') THEN i.round_number END) AS current_round
FROM job_applications a
LEFT JOIN interview_records i ON a.id = i.application_id AND i.deleted = 0
WHERE a.deleted = 0
GROUP BY a.id;
```

### 5.4 状态转换规则类

```java
package com.jobtracker.constants;

import java.util.*;

/**
 * 状态转换规则定义
 */
public class TransitionRules {

    /**
     * 申请状态合法转换映射
     */
    public static final Map<ApplicationStatus, Set<ApplicationStatus>> APPLICATION_TRANSITIONS =
        Map.ofEntries(
            // WISHLIST 可转换到
            Map.entry(ApplicationStatus.WISHLIST,
                Set.of(ApplicationStatus.APPLIED, ApplicationStatus.WITHDRAWN)),

            // APPLIED 可转换到
            Map.entry(ApplicationStatus.APPLIED,
                Set.of(ApplicationStatus.SCREENING, ApplicationStatus.INTERVIEW,
                       ApplicationStatus.REJECTED, ApplicationStatus.WITHDRAWN)),

            // SCREENING 可转换到
            Map.entry(ApplicationStatus.SCREENING,
                Set.of(ApplicationStatus.INTERVIEW, ApplicationStatus.REJECTED,
                       ApplicationStatus.WITHDRAWN)),

            // INTERVIEW 可转换到
            Map.entry(ApplicationStatus.INTERVIEW,
                Set.of(ApplicationStatus.FINAL_ROUND, ApplicationStatus.OFFERED,
                       ApplicationStatus.REJECTED, ApplicationStatus.WITHDRAWN)),

            // FINAL_ROUND 可转换到
            Map.entry(ApplicationStatus.FINAL_ROUND,
                Set.of(ApplicationStatus.OFFERED, ApplicationStatus.REJECTED,
                       ApplicationStatus.WITHDRAWN)),

            // OFFERED 可转换到
            Map.entry(ApplicationStatus.OFFERED,
                Set.of(ApplicationStatus.ACCEPTED, ApplicationStatus.DECLINED,
                       ApplicationStatus.EXPIRED))
        );

    /**
     * 面试状态合法转换映射
     */
    public static final Map<InterviewStatus, Set<InterviewStatus>> INTERVIEW_TRANSITIONS =
        Map.ofEntries(
            Map.entry(InterviewStatus.SCHEDULED,
                Set.of(InterviewStatus.IN_PROGRESS, InterviewStatus.COMPLETED,
                       InterviewStatus.CANCELLED, InterviewStatus.NO_SHOW)),

            Map.entry(InterviewStatus.IN_PROGRESS,
                Set.of(InterviewStatus.COMPLETED, InterviewStatus.CANCELLED)),

            Map.entry(InterviewStatus.COMPLETED,
                Set.of(InterviewStatus.PASSED, InterviewStatus.FAILED)),

            Map.entry(InterviewStatus.CANCELLED,
                Set.of(InterviewStatus.SCHEDULED)),

            Map.entry(InterviewStatus.NO_SHOW,
                Set.of(InterviewStatus.SCHEDULED))
        );
}
```

### 5.3 状态转换服务实现

```java
package com.jobtracker.service.impl;

import com.jobtracker.constants.*;
import com.jobtracker.entity.InterviewRecord;
import com.jobtracker.entity.JobApplication;
import com.jobtracker.exception.InvalidStatusTransitionException;
import com.jobtracker.mapper.InterviewRecordMapper;
import com.jobtracker.mapper.JobApplicationMapper;
import com.jobtracker.service.StatusTransitionService;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.List;

@Slf4j
@Service
@RequiredArgsConstructor
public class StatusTransitionServiceImpl implements StatusTransitionService {

    private final JobApplicationMapper applicationMapper;
    private final InterviewRecordMapper interviewMapper;

    @Override
    public boolean canTransition(ApplicationStatus from, ApplicationStatus to) {
        return from.canTransitionTo(to);
    }

    @Override
    public boolean canTransition(InterviewStatus from, InterviewStatus to) {
        return TransitionRules.INTERVIEW_TRANSITIONS
            .getOrDefault(from, Collections.emptySet())
            .contains(to);
    }

    @Override
    public List<ApplicationStatus> getNextPossibleStatuses(ApplicationStatus current) {
        return TransitionRules.APPLICATION_TRANSITIONS
            .getOrDefault(current, Collections.emptySet())
            .stream()
            .toList();
    }

    @Override
    @Transactional
    public void transitionApplicationStatus(Long applicationId, ApplicationStatus newStatus) {
        JobApplication app = applicationMapper.selectById(applicationId);
        if (app == null) {
            throw new IllegalArgumentException("申请不存在: " + applicationId);
        }

        ApplicationStatus currentStatus = ApplicationStatus.fromCode(app.getStatus());

        // 验证转换是否合法
        if (!canTransition(currentStatus, newStatus)) {
            throw new InvalidStatusTransitionException(
                String.format("不允许从 %s 转换到 %s", currentStatus, newStatus));
        }

        // 执行转换
        app.setStatus(newStatus.getCode());
        app.setUpdatedAt(LocalDateTime.now());
        applicationMapper.updateById(app);

        log.info("申请状态变更: applicationId={}, {} → {}",
            applicationId, currentStatus, newStatus);
    }

    @Override
    @Transactional
    public void transitionInterviewStatus(Long interviewId, InterviewStatus newStatus) {
        InterviewRecord interview = interviewMapper.selectById(interviewId);
        if (interview == null) {
            throw new IllegalArgumentException("面试记录不存在: " + interviewId);
        }

        InterviewStatus currentStatus = InterviewStatus.fromCode(interview.getStatus());

        // 验证转换是否合法
        if (!canTransition(currentStatus, newStatus)) {
            throw new InvalidStatusTransitionException(
                String.format("不允许从 %s 转换到 %s", currentStatus, newStatus));
        }

        // 执行转换
        interview.setStatus(newStatus.getCode());
        interview.setUpdatedAt(LocalDateTime.now());
        interviewMapper.updateById(interview);

        // 联动更新申请状态
        handleInterviewStatusChange(interview, newStatus);

        log.info("面试状态变更: interviewId={}, {} → {}",
            interviewId, currentStatus, newStatus);
    }

    /**
     * 面试状态变更联动处理
     */
    private void handleInterviewStatusChange(InterviewRecord interview, InterviewStatus newStatus) {
        Long applicationId = interview.getApplicationId();

        switch (newStatus) {
            case SCHEDULED -> {
                // 安排面试 → 申请状态变为 INTERVIEW
                JobApplication app = applicationMapper.selectById(applicationId);
                if (ApplicationStatus.fromCode(app.getStatus())
                    .isBefore(ApplicationStatus.INTERVIEW)) {
                    transitionApplicationStatus(applicationId, ApplicationStatus.INTERVIEW);
                }
            }
            case PASSED -> {
                // 面试通过 → 检查是否还有后续面试
                boolean hasMore = hasUpcomingInterviews(applicationId, interview.getId());
                if (!hasMore) {
                    log.info("所有面试已完成，等待最终结果: applicationId={}", applicationId);
                }
            }
            case FAILED -> {
                // 面试未通过
                if (isFinalInterview(interview)) {
                    transitionApplicationStatus(applicationId, ApplicationStatus.REJECTED);
                }
            }
            default -> {}
        }
    }

    private boolean hasUpcomingInterviews(Long applicationId, Long excludeInterviewId) {
        // 查询是否还有其他待处理的面试
        return interviewMapper.countUpcoming(applicationId, excludeInterviewId) > 0;
    }

    private boolean isFinalInterview(InterviewRecord interview) {
        // 判断是否为最后一轮面试（可根据面试类型或顺序判断）
        return "HR".equals(interview.getInterviewType());
    }
}
```

### 5.5 多轮面试服务

```java
package com.jobtracker.service;

import com.jobtracker.dto.InterviewDTO;
import com.jobtracker.entity.InterviewRecord;

/**
 * 面试管理服务（支持多轮面试）
 */
public interface InterviewManagementService {

    /**
     * 安排下一轮面试
     *
     * @param applicationId 申请ID
     * @param interviewType 面试类型
     * @param interviewDate 面试时间
     * @return 新建的面试记录
     */
    InterviewRecord scheduleNextRound(Long applicationId, String interviewType,
                                       LocalDateTime interviewDate);

    /**
     * 获取申请的所有面试（按轮次排序）
     */
    List<InterviewRecord> getInterviewsByApplication(Long applicationId);

    /**
     * 获取当前进行中的面试
     */
    InterviewRecord getCurrentInterview(Long applicationId);

    /**
     * 完成面试并记录结果
     */
    void completeInterview(Long interviewId, String result, Integer rating, String feedback);

    /**
     * 获取面试进度
     */
    InterviewProgress getProgress(Long applicationId);
}

// ============ 实现类 ============

@Service
@RequiredArgsConstructor
public class InterviewManagementServiceImpl implements InterviewManagementService {

    private final InterviewRecordMapper interviewMapper;
    private final JobApplicationMapper applicationMapper;
    private final StatusTransitionService statusTransitionService;

    @Override
    public InterviewRecord scheduleNextRound(Long applicationId, String interviewType,
                                              LocalDateTime interviewDate) {
        // 获取当前轮次数
        int nextRound = interviewMapper.countByApplicationId(applicationId) + 1;

        // 判断是否为终面（HR面通常为终面）
        boolean isFinal = "HR".equals(interviewType);

        InterviewRecord interview = new InterviewRecord();
        interview.setApplicationId(applicationId);
        interview.setInterviewType(interviewType);
        interview.setInterviewDate(interviewDate);
        interview.setRoundNumber(nextRound);
        interview.setIsFinal(isFinal);
        interview.setStatus(InterviewStatus.SCHEDULED.getCode());

        interviewMapper.insert(interview);

        // 触发状态联动
        statusTransitionService.onInterviewScheduled(applicationId);

        return interview;
    }

    @Override
    public void completeInterview(Long interviewId, String result, Integer rating,
                                   String feedback) {
        InterviewRecord interview = interviewMapper.selectById(interviewId);

        // 更新面试结果
        interview.setStatus(InterviewStatus.COMPLETED.getCode());
        interview.setResult(result);
        interview.setRating(rating);
        interview.setFeedback(feedback);

        interviewMapper.updateById(interview);

        // 根据结果更新状态
        InterviewStatus newStatus = "PASSED".equals(result)
            ? InterviewStatus.PASSED
            : InterviewStatus.FAILED;

        statusTransitionService.transitionInterviewStatus(interviewId, newStatus);
    }
}
```

## 7. 实施建议

### 7.1 分阶段实施

| 阶段 | 内容 | 风险 |
|------|------|------|
| Phase 1 | 添加 `InterviewStatus` 枚举，保持 `ApplicationStatus` 不变 | 低 |
| Phase 2 | 实现状态转换验证服务 | 低 |
| Phase 3 | 实现面试-申请状态联动 | 中 |
| Phase 4 | 扩展 `ApplicationStatus`（可选） | 高（需数据迁移） |

### 7.2 建议

1. **优先实现联动**：先让现有状态自动联动，体验提升明显
2. **保持向后兼容**：新状态作为扩展，不破坏现有逻辑
3. **审计日志**：记录所有状态变更，便于追溯
4. **前端配合**：状态变更后刷新看板视图
